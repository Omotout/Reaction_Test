# 仮説階層と統計解析プラン

> EMS訓練の反応時間短縮効果を、**観測指標（RT・正答率）** と **DDMによる機構分解** の二段階で検証する設計書。
> `EXPERIMENT_V3_DESIGN.md` の補足として、教授・査読者・自分自身に対して **「何を、どう、なぜそう判断するか」** を明示する。

## 1. 研究全体の問い

EMSを「主体感（Agency）を維持できるギリギリのタイミング」で発火する訓練は、訓練後の反応時間をどう変えるか。

- **第一の問い（What）**: 訓練効果（RT短縮）は EMS群で通常群より大きいか？
- **第二の問い（How）**: もし RT 短縮があったとして、それは反応プロセスのどの段階で起きているのか？

これら二つは**独立した問い**であり、それぞれ別の手法で検証する。

---

## 2. 仮説の階層

### H1（主仮説 — 観測指標）

| | 仮説 | 期待される結果 |
|---|---|---|
| **H1a** | EMS群の ΔRT は通常群より大きい | ΔRT_AgencyEMS < ΔRT_Voluntary |
| **H1b** | EMS群の正答率は通常群と同等以上 | ΔAccuracy_AgencyEMS ≥ ΔAccuracy_Voluntary |

**H1a は主検定**。H1b は H1a を補強する制約条件（速度を上げるために正答率を犠牲にしていないことの確認）。

### H2（機構仮説 — DDMによる分解、探索的）

H1 が支持された場合のみ意味を持つ。**RT短縮があったとして、それは DDM のどのパラメータの変化で説明されるか**。

| | 仮説 | 期待される結果 | 立場 |
|---|---|---|---|
| **H2a** | 群間で Δa（決定閾値）に差がない | HDIが0をまたぐ・ROPE内 | 仮説的（トレードオフ否定） |
| **H2b** | EMS群で Δt（非決定時間）が短縮 | Δt_AgencyEMS < Δt_Voluntary | 仮説的（運動プライミング） |
| **H2c** | EMS群で Δv（ドリフト率）が変化 | 方向性は事前指定なし | 探索的 |
| **H2d** | EMS群で Δ決定時間 が変化 | 方向性は事前指定なし | 探索的 |

ここで **Δ決定時間** は a と v から計算される導出量：
```
Decision Time = (a / 2v) · tanh(av / 2)
```

H2b と H2d は**互いに排他的でない**。両方変化していれば「運動段階と決定段階の両方が訓練で改善した」という解釈になる。

### H1 と H2 の関係

```
H1 が支持      → H2 を検討する意義がある（機構分解）
H1 が不支持    → H2 は探索的観察にとどめる（効果がないものの分解は意味薄）
H1b が不支持   → 速度-正答率トレードオフの可能性 → H2a で確認
```

## 3. 解析戦略

### Step 1: H1 の検定（HDDM不要）

**スクリプト**: `analyze_training_effect.py`（EZ-DDM版に既存実装あり、HDDM不要部分を主検定として参照）

**手法**:
| 指標 | 検定 |
|---|---|
| ΔRT (ms) | 独立2群 Welch t 検定（片側） + ベイズ推定（BEST法または BF10） |
| ΔAccuracy | 独立2群 Welch t 検定（片側または同等性検定） |

**報告内容**:
- 各群の Δ の平均・SD・95% CI
- 群間差の効果量（Cohen's d）
- p 値（参考）と事後確率（主指標）

**判定基準**:
- 強い証拠: P(ΔRT_EMS < ΔRT_Voluntary | data) ≥ 0.95
- 中程度: 0.90–0.95
- 弱い: 0.85–0.90
- 証拠なし: < 0.85

### Step 2: H2 の機構分解（HDDM使用）

**スクリプト**: `analyze_training_effect_hddm.py`

**手法**: 階層ベイズドリフト拡散モデル（HDDM, Wiecki et al. 2013）

```python
HDDMStimCoding(
    stim_col='stim',           # Left / Right
    split_param='v',
    depends_on={
        'v': 'Phase',           # Baseline / PostTest
        'a': 'Phase',
        't': 'Phase',
    },
    p_outlier=0.05,
)
```

**解析手順**:
1. 群ごとに別モデルでフィット（被験者間デザインを尊重）
2. 各群について Δv, Δa, Δt の事後分布を取得
3. Δ決定時間を a と v の事後サンプルから派生計算
4. 群間で各 Δ の事後分布を比較
5. 寄与率を派生計算: `frac_t = Δt / ΔRT`, `frac_decision = Δ決定時間 / ΔRT`

**判定基準**: H1 と同じ事後確率の階層（0.95/0.90/0.85）+ HDIによる ROPE 比較。

## 4. 結果パターンと解釈

| H1a | Δa | Δt | Δv | 解釈 |
|---|---|---|---|---|
| ✓ | ≈0 | <0 | ≈0 | **運動プライミング**（仮説通り、最も期待される結果） |
| ✓ | ≈0 | ≈0 | >0 | **意思決定促進**（EMS が知覚-運動連合を強化） |
| ✓ | ≈0 | <0 | >0 | **複合効果**（運動と決定の両方が改善） |
| ✓ | <0 | ? | ? | **トレードオフ警告**（速度-正答率を犠牲にした可能性 → H1b と併せて要確認） |
| ✓ | ≈0 | 局在不明 | 局在不明 | **効果あるが機構未特定**（試行数・被験者数不足の可能性） |
| ✗ | — | — | — | **効果なし**（ROPE による equivalence 主張、または検出力不足） |

「✓」= P > 0.95 で群間差あり、「<0」= P > 0.95 で減少、「≈0」= HDI が 0 を含み ROPE 内、「?」= 解釈保留。

---

## 5. 多重比較補正

H1 と H2 で**別々に扱う**（gated analysis）。

- **H1（2検定）**: H1a と H1b。これは前段の主検定。Bonferroni または FDR を適用してもよいが、検定数が少ないため未補正でも実用上問題ない。
- **H2（4検定）**: H2a, H2b, H2c, H2d。H1 が支持された場合のみ実施。BH-FDR 補正を適用。

H1 と H2 を混合して 6 検定一括補正すると、主仮説のパワーが探索的検定の存在で penalty を受けるため**避ける**。

---

## 6. パワー分析と必要サンプル数

**観測指標（H1a）の検出**:
- 想定効果量 Cohen's d ≈ 0.6–0.8（中〜大）
- α=0.05、Power=0.80 → 各群 18 名 × 2 群 = **36 名**
- 既存設計（V3）はこれを満たす

**機構分解（H2b: Δt の検出）**:
- 効果量と被験者間個人差に強く依存
- σ_t_between = 20 ms 仮定: Δt = 10 ms 検出に各群 50 名以上
- σ_t_between = 10 ms 仮定: 各群 20 名で十分
- **本実験のサンプルサイズは H1 のために設計されており、H2 は探索的位置付け**

**推奨**: パイロット 5 名で σ_t_between を実測し、H2 検出に必要なサンプル数を再計算。

## 7. 事前登録（Pre-registration）の論点

将来的な事前登録を見据えて、以下を**仮説駆動 (confirmatory)** と **探索的 (exploratory)** に明確に区別する。

### Confirmatory（仮説駆動）
- H1a: ΔRT の群間比較（片側）
- H1b: ΔAccuracy の同等性または非劣性
- H2a: Δa の群間で差がないこと（ROPE による equivalence）
- H2b: Δt の群間比較（片側）

### Exploratory（探索的）
- H2c: Δv の群間比較（方向性事前指定なし）
- H2d: Δ決定時間 の群間比較
- frac_t, frac_decision の群比較
- 個人レベルの reference distribution 分析（Voluntary 群分布での EMS 群被験者の percentile）
- 主体感（Agency）の連続値スコアと Δt の関連分析

## 9. 教授・査読者向け FAQ

### Q1. なぜ平均RT比較だけでなくHDDMを使うのか？
A. EMSの効果が運動段階（t）に出るのか、意思決定段階（v, a）に出るのかは、平均RTだけでは区別できない。本研究の主目的の一つは**効果の局在を明らかにする**ことであり、これにはRT分布全体を要素分解する DDM が必要。

### Q2. なぜ階層ベイズか？
A. 各被験者を独立に DDM 推定すると、試行数が少ない被験者の推定が不安定になる。階層ベイズの部分プーリングはこれを群レベル分布で安定化する。また、点推定ではなく事後分布が得られるため、不確実性を明示的に扱える。

### Q3. Δt が出なかったらどうするのか？
A. ベイズ推定では「効果なし」を ROPE による equivalence として積極的に主張できる。Δt の HDI が ±2 ms の ROPE 内に収まれば、「実質的に Δt は変化していない」と結論できる。さらに Δv や Δ決定時間 が変化していれば、それは EMS が運動段階ではなく意思決定段階に作用したという発見になる（仮説外だが価値ある知見）。

### Q4. 決定時間も含めて見ているのか？
A. 含めている。Δa と Δv を別々に推定するだけでなく、Δ決定時間 = (a/2v)·tanh(av/2) を事後サンプルから派生計算し、群間比較する。これにより、たとえば「v は上がったが a も上がって結局決定時間は変わらない」というケースも見逃さない。

### Q5. Δa が下がっていたらトレードオフではないか？
A. その懸念は重要。Δa が群間で異なる場合、H2a が棄却され「速度-正答率トレードオフ」の可能性が高い。これは H1b（正答率の維持）と併せて評価する。Δa が変化していて正答率が下がっているなら、それは訓練効果ではなく単なる「雑な反応」であり、研究の結論にも反映する。

### Q6. 被験者間デザインの理由は？
A. 訓練効果の比較が目的のため。被験者内で「EMS訓練 → ウォッシュアウト → 通常訓練」とすると、EMS訓練の carry-over（運動表象の長期残存）と練習効果が混入し、純粋な比較が困難。被験者間にすることで各被験者は1種類の訓練のみを経験し、訓練法そのものを比較できる。代償として個人差が群間差に混入するため、サンプルサイズで対応する。

### Q7. なぜ各群18名？
A. H1a（ΔRTの群間比較）について、効果量 f=0.3、α=0.05、Power=0.80 で算出した値。H2 については探索的位置付けのため、本サンプルサイズで仮に検出力が不足しても本研究の主目的（H1）には影響しない。パイロット結果で σ_t_between を実測した上で、必要に応じて追加被験者を検討する。
