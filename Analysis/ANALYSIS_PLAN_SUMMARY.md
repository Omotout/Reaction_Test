# 解析プラン 1ページサマリ

> 教授・ミーティング向け。詳細は `HYPOTHESIS_AND_ANALYSIS_PLAN.md` 参照。

## 研究の問い

**EMS訓練は通常訓練より反応時間（RT）を短縮するか。短縮があるとして、それは反応プロセスのどの段階で起きているか。**

## 仮説の階層

```
┌─────────────────────────────────────────────────────────────────┐
│  H1: 観測指標の比較（主仮説）                                   │
│  ─────────────────────────────────                              │
│   H1a: EMS群の ΔRT > 通常群の ΔRT （より大きな短縮）            │
│   H1b: EMS群の正答率は通常群と同等以上                          │
│  → 手法: 独立2群 t 検定 + ベイズ推定                            │
│  → HDDMは使わない                                               │
└─────────────────────────────────────────────────────────────────┘
                            │
                            ↓ H1 が支持された場合
┌─────────────────────────────────────────────────────────────────┐
│  H2: DDM による機構分解（探索的）                               │
│  ─────────────────────────────────                              │
│   H2a: Δa（決定閾値）に群間差なし → トレードオフ否定（仮説的）  │
│   H2b: Δt（非決定時間）が EMS群で短縮 → 運動プライミング（仮説的）│
│   H2c: Δv（ドリフト率）の変化（探索的）                         │
│   H2d: Δ決定時間 = Δ((a/2v)·tanh(av/2)) の変化（探索的）        │
│  → 手法: 階層ベイズドリフト拡散モデル（HDDM）                   │
│  → 群ごとに別フィット、事後分布を群間比較                       │
└─────────────────────────────────────────────────────────────────┘
```

## 判断基準（事後確率）

| P(Δ < 0 \| data) | 解釈 | 頻度論との対応 |
|---|---|---|
| ≥ 0.95 | 強い証拠 (strong) | p < 0.05 相当（片側） |
| 0.90–0.95 | 中程度 (moderate) | p ≈ 0.05–0.10 |
| 0.85–0.90 | 弱い (anecdotal) | trend |
| < 0.85 | 証拠なし | n.s. |

加えて HDI（95% 信用区間）と ROPE（実質的同等領域、Δt なら ±2 ms）による equivalence 判定も併用。

## なぜこの設計か

**Q: なぜ HDDM を使うのか？**
A. 平均RTだけ見ると、その短縮が「運動段階（t）の効率化」なのか「意思決定段階（v, a）の促進」なのか区別できない。EMSの効果が運動プライミングである、という仮説の検証には**反応時間を要素分解**する必要がある。

**Q: なぜ階層ベイズか？**
A. 部分プーリングによる推定の安定化と、不確実性を事後分布として明示できる利点。Wiecki et al. (2013) の HDDM パッケージが業界標準。

**Q: 決定時間も含めるのか？**
A. 含める。Δa と Δv だけでなく、a と v の事後サンプルから Δ決定時間 を派生計算し、群間比較する。「v は上がったが a も上がって決定時間は変わらない」という相殺ケースを見逃さない。

**Q: 効果が検出されなかったらどうする？**
A. ベイズ推定なので「効果なし」も積極的に主張できる。HDI が ROPE 内に収まれば equivalence、HDI が広ければ検出力不足、と区別する。
