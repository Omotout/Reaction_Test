# 結果解釈フロー（Decision Tree）

> 結果が出てから「どう解釈するか」で迷わないための、事前定義の判断フロー。
> `HYPOTHESIS_AND_ANALYSIS_PLAN.md` の運用版。

## 用語

| 表記 | 意味 |
|---|---|
| ✓ | 強い証拠あり（事後確率 ≥ 0.95、または HDI が 0 をまたがない） |
| △ | 中程度（0.85–0.95） |
| ✗ | 証拠なし（< 0.85）または ROPE 内 |
| ROPE | Region of Practical Equivalence。Δt なら ±2 ms、Δa なら ±0.05 など事前指定 |

---

## 第1段階: H1（観測指標）の判定

### Step 1.1 ΔRT の群間比較

```
P(ΔRT_AgencyEMS < ΔRT_Voluntary | data) を計算
  ├─ ≥ 0.95  →  H1a 強く支持。Step 1.2 に進む
  ├─ ≥ 0.90  →  H1a 中程度に支持。Step 1.2 に進む（証拠の弱さは結論に反映）
  ├─ ≥ 0.85  →  H1a 弱く支持。"trend" として報告
  └─ < 0.85  →  H1a 不支持。Step E1（null result 解釈）へ
```

### Step 1.2 ΔAccuracy の群間比較

```
P(ΔAccuracy_AgencyEMS ≥ ΔAccuracy_Voluntary − ε | data) を計算（同等性）
  ├─ 同等以上  →  H1b 支持。Step 2 へ進む（機構分解）
  └─ EMS群で正答率低下  →  ⚠ 速度-正答率トレードオフの懸念
                          →  Step 1.3 で Δa を確認してから判断
```

### Step 1.3 トレードオフ確認（H1b で懸念があった場合のみ）

```
P(Δa_AgencyEMS < Δa_Voluntary | data) を計算
  ├─ ≥ 0.95  →  Δa が EMS 群で大きく低下 → トレードオフが原因
              →  「EMS は決定閾値を下げて速度を稼いだが、正答率を犠牲にした」
              →  訓練効果としての主張は弱める
  └─ < 0.95  →  Δa の群間差は明確でない
              →  正答率低下は別の理由（疲労、不注意等）の可能性
              →  H1 の結論を保留、機構分解は探索的に実施
```

---

## 第2段階: H2（DDM分解）の判定

H1a 支持（≥ 0.85）かつ H1b 問題なしの場合のみ実施。

### Step 2.1 Δt（非決定時間）の群間比較

```
P(Δt_AgencyEMS < Δt_Voluntary | data) を計算
  ├─ ≥ 0.95  →  H2b 支持「運動プライミング効果」
  ├─ 0.85–0.95  →  H2b 弱く支持
  ├─ HDI が ROPE(±2ms) 内  →  Δt は実質的に変化なし
  └─ それ以外  →  Δt は判定保留
```

### Step 2.2 Δa（決定閾値）の群間比較（同等性）

```
HDI(Δa_AgencyEMS − Δa_Voluntary) が ROPE(±0.05) 内に収まるか
  ├─ 収まる  →  H2a 支持「閾値変化なし、トレードオフ否定」
  ├─ 0 をまたぐが ROPE をはみ出す  →  「証拠不十分、効果は不明」
  └─ 0 をまたがない  →  H2a 不支持「Δa にも変化あり、要追加解釈」
```

### Step 2.3 Δv（ドリフト率）の群間比較（探索的）

```
P(Δv_AgencyEMS > Δv_Voluntary | data)
  ├─ ≥ 0.95  →  EMS 群でドリフト率上昇「意思決定促進」
  ├─ ≤ 0.05  →  EMS 群でドリフト率低下（要解釈）
  └─ それ以外  →  群間差は明確でない
```

### Step 2.4 Δ決定時間 の群間比較（探索的）

```
Δ決定時間 = Δ((a/2v)·tanh(av/2))  を事後サンプルから派生計算
P(Δ決定時間_AgencyEMS < Δ決定時間_Voluntary | data)
  ├─ ≥ 0.95  →  決定段階の処理時間が短縮
  └─ < 0.95  →  決定段階の変化は明確でない
```

### Step 2.5 寄与率の確認

```
被験者ごとの frac_t = Δt / ΔRT, frac_decision = Δ決定時間 / ΔRT を計算
  群平均で frac_t + frac_decision ≈ 1 になっているはず（観測 RT 短縮の説明）

  AgencyEMS 群で:
    frac_t  >> frac_decision  →  非決定時間主導の短縮
    frac_t  << frac_decision  →  決定時間主導の短縮
    どちらも ≈ 0.5            →  両者の複合
```

## 結果パターン別の論文ストーリー

### Pattern A: 仮説通りの「運動プライミング」
```
H1a ✓, H1b ✓, H2a ✓ (Δa不変), H2b ✓ (Δt短縮), H2c ✗ (Δv不変)
```
**ストーリー**: EMS訓練は運動段階のプライミングにより RT を短縮する。Kasahara et al. (2021) を支持。

### Pattern B: 「意思決定促進」（仮説外だが価値あり）
```
H1a ✓, H1b ✓, H2a ✓ (Δa不変), H2b ✗ (Δt不変), H2c ✓ (Δv上昇)
```
**ストーリー**: EMS は単なる運動経路ではなく、知覚-運動連合の強化を介して意思決定そのものを促進する。Kasahara et al. の解釈に新たな機構的視点を追加する貢献。

### Pattern C: 「複合効果」
```
H1a ✓, H1b ✓, H2a ✓, H2b ✓ (Δt短縮), H2c ✓ (Δv上昇)
```
**ストーリー**: EMS の効果は単一段階ではなく、運動段階と決定段階の両方に作用する。frac_t と frac_decision で寄与の比率を議論。

### Pattern D: 「トレードオフ警告」
```
H1a ✓, H1b ✗ (正答率低下), H2a ✗ (Δa低下)
```
**ストーリー**: EMS群は反応が速くなったが、決定閾値を下げて正答率を犠牲にした。これは訓練効果ではなく**速度バイアス**であり、メカニズムとしての主張は慎重に行う。Bogacz et al. (2010) の枠組みで議論。

### Pattern E: 「効果検出されず」
```
H1a ✗
```
**ストーリー**: 本パラダイムでは EMS 訓練の効果は検出されなかった。
- ROPE 内 → 「実質的に効果なし」と equivalence 主張
- HDI 広い → 「検出力不足、より大規模研究が必要」と結論

いずれにせよ、Kasahara et al. の追試として価値ある null result。Δt や Δv の事後分布は探索的に提示。

### Pattern F: 「機構不明だが効果あり」
```
H1a ✓, H1b ✓, H2a ✓, H2b ?, H2c ?, H2d ?
```
**ストーリー**: RT は短縮したが DDM パラメータレベルでは局在を特定できなかった。サンプルサイズの限界、または効果が複数パラメータに分散している可能性。今後の研究課題として明示。

---

## 検出力不足の見極め

H2 の検出が曖昧な場合、それが **本当に効果がない** のか **検出力不足** なのかを区別する：

```
事後分布の HDI 幅を確認:
  HDI が 0 をまたぎ、かつ HDI 幅 > 2 × ROPE
    → 検出力不足。「効果は不明、より大きな N が必要」
  HDI が 0 をまたぎ、かつ HDI 幅 ≤ 2 × ROPE
    → 効果なしと積極的に主張可能
```

---

## 報告すべきもの（最低限）

各検定について以下を報告：

1. 各群の Δ の **事後平均**
2. **95% HDI**
3. **事後確率** P(Δ < 0 | data) または P(差 < 0 | data)
4. **ROPE** との関係（HDI が ROPE 内/外/またぐ）
5. **観測値**（RT は ms、accuracy は %、t は ms）
6. **n**（各群の有効被験者数）

可能なら追加で：

7. 効果量（Cohen's d 相当、被験者レベル Δ から計算）
8. ベイズファクター（BF10、Savage-Dickey 法）
9. 被験者レベルの個別プロット（reference distribution）

---

## 報告文の例

### Pattern A の報告例

> AgencyEMS 群（n=18）と Voluntary 群（n=18）の比較において、
> ΔRT は EMS 群で有意に大きく短縮した
> （ΔRT_EMS = -23.4 ms, ΔRT_Voluntary = -8.1 ms,
> 群間差 = -15.3 ms, 95% HDI [-22.1, -8.5],
> P(EMS larger reduction | data) = 0.998）。
> 正答率は両群で同等であり（H1b 支持）、速度-正答率トレードオフは観察されなかった。
>
> HDDM による分解では、Δt（非決定時間）の事後分布が EMS 群で有意に大きく短縮し
> （Δt_EMS = -18.2 ms, Δt_Voluntary = -3.1 ms,
> 群間差 = -15.1 ms, 95% HDI [-21.8, -8.4],
> P(EMS larger reduction | data) = 0.997）、
> 一方で Δa の HDI（[-0.04, +0.03]）は ROPE(±0.05) 内に収まり、
> 決定閾値の変化は支持されなかった。
> 被験者レベルの寄与率は frac_t = 0.78 (median)、
> 観測された RT 短縮の大部分が非決定時間の短縮で説明された。
>
> これらの結果は、EMS 訓練が運動準備段階の処理を効率化することで RT を短縮するという、
> Kasahara et al. (2021) の仮説を支持する。

### Pattern E の報告例

> AgencyEMS 群（n=18）と Voluntary 群（n=18）の比較において、
> ΔRT の群間差は明確には支持されなかった
> （群間差 = -3.2 ms, 95% HDI [-12.1, +5.8], P(EMS larger reduction | data) = 0.76）。
> HDI 幅は約 18 ms と広く、ROPE(±5 ms) を大きく超えており、
> 効果なしを積極的に主張するには検出力不足である。
>
> 探索的に HDDM 分解を行ったところ、Δt の群間差も曖昧であった
> （群間差 = -2.8 ms, 95% HDI [-10.3, +5.1]）。
>
> 本研究では Kasahara et al. (2021) の効果を再現するに至らなかった。
> サンプルサイズ（各群18名）が当初の効果量見積もり（Cohen's d ≈ 0.6）に基づいていたが、
> 実際の効果量はこれより小さい可能性、または個人差が想定より大きい可能性がある。
> 追加データ収集または被験者内デザインでの再検証が今後の課題である。
